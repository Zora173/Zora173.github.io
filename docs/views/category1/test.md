2. 按下 `alt` + `/` 键
3. 出现 `vp_doc` 回车
4. 光标出现直接输入标题，会同步输入描述和关键词
5. 按下 `tab` 光标到 `tag` 那里，输入标签
6. 分类会自动取当前文件夹名称，可以自己后面改
   好的
